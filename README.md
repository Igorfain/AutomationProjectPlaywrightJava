# **Java Playwright Automation Project**

A professional automation framework built in **Java + Maven**, combining both **UI** and **API** testing.
Created for **self-training**, portfolio purposes, and to demonstrate real test automation skills using modern industry practices.

This project is designed for **self-training** and continuous skill improvement in automated testing.
It demonstrates:

* **building a structured automation framework**
* **combining UI and API tests within one project**
* **clean and reusable architecture**
